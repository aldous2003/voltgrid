PROJECT OVERVIEW

What is VoltGrid?

VoltGrid is an embedded IoT system that provides prepaid electricity metering for multi-room environments. Residents load credits via coin slot, and the ESP32 monitors real-time consumption — automatically cutting off the relay when credit runs out.

⚡

Real-Time Metering

PZEM-004T measures voltage, current, power, frequency and energy every second.

💳

Coin-Based Top-Up

Residents insert coins to load credit. The system tracks balance and remaining kWh.

🔁

Relay Control

Electricity is automatically disconnected when credit reaches zero. Manual override available.

📡

Firebase Sync

All data is published to Firebase in real-time, powering this monitoring dashboard.

🖥️

LCD + Keypad UI

On-device interface lets users check balance, select rooms, and interact with the system.

📊

Live Analytics

Charts show power vs time, daily and weekly energy consumption for each room.
